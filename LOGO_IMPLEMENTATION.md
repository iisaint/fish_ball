## 🔍 技術細節

### 圖片路徑
- **開發環境**: `/wan_dong_logo.jpg`
- **生產環境**: `/wan_dong_logo.jpg`
- Vite 會自動處理 public 資料夾的檔案

### CSS 類別說明
- `h-20`: 高度 5rem (80px)
- `h-28`: 高度 7rem (112px) - 僅在 md 斷點以上
- `w-auto`: 寬度自動調整保持比例
- `object-contain`: 保持圖片原始比例，不裁切
- `md:h-28`: 在中等螢幕以上使用較大尺寸

### RWD 響應式
- **手機** (<768px): Logo 高度 80px
- **平板/桌面** (≥768px): Logo 高度 112px
- 寬度自動縮放，保持圖片比例

## 🔧 自訂調整

### 如果想調整 Logo 大小

**主頁面標題**：
```jsx
// 改變這些類別
className="h-20 md:h-28"  // 目前：80px / 112px

// 更大
className="h-24 md:h-32"  // 96px / 128px

// 更小
className="h-16 md:h-20"  // 64px / 80px
```

**訂購單預覽**：
```jsx
// 改變這個類別
className="h-20"  // 目前：80px

// 更大
className="h-24"  // 96px

// 更小
className="h-16"  // 64px
```

### 如果想使用不同的 Logo

1. 將新 logo 放到 `public/` 資料夾
2. 在 `src/App.jsx` 中更新圖片路徑：
```jsx
src="/your-new-logo.jpg"
```
